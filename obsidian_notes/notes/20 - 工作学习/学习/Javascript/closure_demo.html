💡 基础概念：什么是闭包？

闭包是函数和其词法环境的组合。内部函数可以访问外部函数的变量，即使外部函数已经执行完毕。

function createGreeting() { let message = "Hello from closure!"; // 外部函数的变量 return function() { console.log(message); // 内部函数访问外部变量 }; } const greet = createGreeting(); greet(); // 输出: "Hello from closure!"

🔢 经典示例：计数器

闭包最经典的应用就是创建私有变量。每次调用返回的函数时，都会访问同一个变量。

function createCounter() { let count = 0; // 私有变量 return function() { count++; // 每次调用都会增加 return count; }; } const counter = createCounter();

⚠️ 常见陷阱：循环中的闭包

在循环中创建闭包时，要注意变量的作用域问题。

// 错误示例 - 所有函数都会输出 3 for (var i = 0; i < 3; i++) { setTimeout(function() { console.log(i); // 都输出 3 }, 100); } // 正确示例 - 使用 let 或 IIFE for (let i = 0; i < 3; i++) { setTimeout(function() { console.log(i); // 输出 0, 1, 2 }, 100); }
