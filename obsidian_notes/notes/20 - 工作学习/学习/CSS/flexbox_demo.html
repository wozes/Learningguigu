当前 CSS 属性：

.flex-container {
  display: flex;
  flex-direction: row;
  justify-content: flex-start;
  align-items: stretch;
  flex-wrap: nowrap;
}

💡 常见应用场景

1. 水平垂直居中

设置 justify-content: center 和 align-items: center 可以实现完美的水平垂直居中。

2. 等间距分布

使用 justify-content: space-between 让第一个和最后一个项目贴边，中间项目等间距分布。

3. 响应式布局

结合 flex-wrap: wrap 可以创建响应式的卡片布局。

4. 垂直布局

设置 flex-direction: column 可以创建垂直方向的布局。
